# 视图生成器

**AI 图片 + 视频一体化商品视觉内容生成平台**

面向电商场景，将 AI 图片生成与 AI 视频生成深度整合为一体。只需输入商品描述，即可自动生成多角度商品图，在无限画布上自由编辑排版，再一键生成商品展示视频——覆盖从主图、详情页素材到短视频的完整视觉链路。

## 为什么选择视图生成器

传统电商视觉制作需要拍摄、P图、剪辑多个工具来回切换。视图生成器把**图片生成 + 画布编辑 + 视频生成**三个环节串联在一个工具里，基于同一个商品项目输出多种视觉素材，满足不同渠道的投放需求：

- 🛒 **电商主图 / 多角度展示** — 正面、侧面、背面、顶视、细节一键生成
- 📱 **商品短视频** — 多角度关键帧 AI 合成，支持自定义过渡描述控制镜头变化
- 📋 **详情页素材** — 画布标注连线 + 文字说明，生成带标注的商品解说图
- 📄 **项目导出 / 备份** — 完整项目 JSON 导出，团队共享或二次编辑

## 功能

### 🎨 AI 图片生成
- 文生图 / 图生图双模式，I2I 优先 + T2I 智能降级
- 支持 5 个标准电商角度：正面 / 侧面 / 背面 / 顶视 / 细节
- 商品描述自由填写，细节越丰富生成越精准

### 🎬 AI 视频生成
- 多张图片作为关键帧，AI 自动生成平滑过渡视频
- **过渡描述自定义**——每两帧之间可写提示词，精细控制画面变化（如「特写鞋底材质」「镜头缓慢旋转」）
- 异步生成 + 轮询状态，不阻塞操作

### 🖼️ 无限画布编辑
- 基于 Fabric.js 的无限画布，缩放 / 平移 / 拖拽图片
- 添加文字、切换背景色
- **标注连线**——从商品图引出贝塞尔曲线 + 文字标签，制作详情页风格素材
- Ctrl + 滚轮缩放，工具栏实时显示缩放比例

### 💾 本地优先
- localStorage 持久化项目，无需服务器、无需登录
- 项目导出为 JSON 文件，可备份、迁移、分享
- API Key 仅在本地内存，不存储不上传

## 快速开始

```bash
git clone https://github.com/hh2079/product-image-generator.git
cd product-image-generator

# 安装依赖
npm install && cd client && npm install && cd ../server && npm install
cd ..

# 启动
npm run dev
```

打开 `http://localhost:5173`

## 使用流程

| 步骤 | 操作 |
|------|------|
| 1. 设置 Key | 点击右上角 **API Key**，填入 Agnes AI Key（[获取](https://apihub.agnes-ai.com)） |
| 2. 描述商品 | 输入商品描述，如「白色平板鞋 帆布材质 圆形鞋头」 |
| 3. 生成图片 | 上传主图，选择角度，AI 生成多角度商品图 |
| 4. 画布编辑 | 拖拽图片到画布，加文字、标注线、换背景 |
| 5. 生成视频 | 选 2+ 张图片，设置过渡描述，一键生成商品视频 |

## 技术栈

| 层 | 技术 |
|---|------|
| 前端 | React 18 + TypeScript + Fabric.js 6 + Zustand + Vite |
| 后端 | Express + TypeScript |
| AI 图片 | Agnes AI `agnes-image-2.0-flash`（T2I / I2I） |
| AI 视频 | Agnes AI `agnes-video-v2.0`（关键帧动画） |

## 适用场景

| 场景 | 说明 |
|------|------|
| 电商多角度展示 | 生成商品各角度图，用于主图/附图 |
| 商品短视频 | 多角度合成展示视频，用于详情页/短视频平台 |
| 详情页素材 | 标注线+文字说明，制作商品解说图 |
| 设计师协作 | 项目 JSON 导出，团队共享素材和排版 |
